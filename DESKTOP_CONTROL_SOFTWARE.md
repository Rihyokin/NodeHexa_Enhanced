# Desktop Control Software Development - 桌面控制软件开发文档

> 本文档记录 Hexapod Smart System 桌面端控制软件的开发进展和技术架构。

---

## 1. 项目概述

### 1.1 目标
开发一款功能强大的桌面端控制软件，用于替代或增强现有的Web控制界面，提供更丰富的功能和更好的用户体验。

### 1.2 主要特性
- 跨平台支持 (Windows/Linux/macOS)
- 实时姿态3D可视化
- 高级运动控制和轨迹规划
- 日志记录和数据采集
- 多设备管理
- Python原型 + C++性能版本

---

## 2. 技术架构

### 2.1 双版本策略

| 版本 | 语言 | 特点 | 适用场景 |
|------|------|------|----------|
| Python版 | Python 3.8+ | 开发快、易调试、功能丰富 | 快速原型、功能验证 |
| C++版 | C++17 | 性能优、响应快、可独立运行 | 产品化、高性能需求 |

### 2.2 通信接口

```
桌面软件 ←→ UART2/蓝牙/WiFi ←→ ESP32固件
```

**通信协议**：
- UART2串口指令（主要）
- 蓝牙BLE（备用）
- WiFi WebSocket（远程控制）

### 2.3 依赖库

**Python版本**：
- PyQt5/PySide6 - GUI框架
- pyserial - 串口通信
- numpy - 数据处理
- matplotlib - 数据可视化
- pybluez - 蓝牙通信（Linux）

**C++版本**：
- Qt6 - GUI框架
- Boost.Asio - 串口/网络通信
- Eigen - 数学计算
- OpenCV - 图像处理

